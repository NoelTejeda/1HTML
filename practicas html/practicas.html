Inicios de creación de una página html

He decido colocar lo poco que he aprendido en esta página a manera de práctica, así para futuras consultas ya tendré una guia


Doctype:
se usa para definir la version del html a usar en este caso seria html5

Significado del lang:
es el estilo del lenguaje que se va a utilizar en la página, por ejemplo: lang="es" (para español) lang="en" (para ingles)

Contenido del tag head:

mata charset="utf-8":
Es la codificación utilizada para indicarle a la página que se usaran caracteres especiales como acentos, la letra ñ, etc. Y es importante tomar en cuenta el tipo de lenguaje definido al inicio (lang)

tile:
Se coloca dentro del head y es la que le coloca a la pestaña en el navegador un titulo

Estilo de texto

se usan los tag "b" y "strong" para colocar el texto en el estilo negrita,
Para hacer el texto con estilo cursiva se usa el tag "i"
texto como borrado se usa el tag "del"
